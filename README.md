# appropriate/nc
Alpine-based image with just nc (via the netcat-openbsd package)

## Usage

```console
$ docker run -it --rm appropriate/nc 127.0.0.1 80 < input
```

## Tags

* `appropriate/nc:latest`: based on `alpine:latest`
* `appropriate/nc:edge`: based on `alpine:edge`
